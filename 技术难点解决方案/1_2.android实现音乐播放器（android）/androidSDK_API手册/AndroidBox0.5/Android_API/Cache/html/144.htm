概述

当用户执行了互动或者触发了一些行为的接口，例如缩放。

 

公共方法

public abstract void onVisibilityChanged (boolean visible)

当缩放控件的显示发生变化时调用

参数

visible  缩放控件是否可见  

 

public abstract void onZoom (boolean zoomIn)

当该控件的拥有者需要被缩放时候调用

参数

zoomIn  缩放的方向：true是放大，false是缩小
